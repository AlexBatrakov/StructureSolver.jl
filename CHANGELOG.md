# Changelog

All notable changes to this project will be documented in this file.

The format is based on *Keep a Changelog*, and this project aims to follow *Semantic Versioning*.

## [0.1.2] - 2026-02-01

- Recruiter-friendly README (quick links, demo snippet) + embedded figures
- Expanded docs (notation/outputs page)
- Release automation (TagBot) and dependency automation (CompatHelper)
- Optional CI coverage upload (non-blocking)

## [0.1.1] - 2026-02-01

- Documentation site (Documenter) with multiple pages and tutorials
- CI workflows for tests and docs
- Stable default tabulated EoS (`Table_EoS`) with experimental Hermite version kept separate
- Optional plotting (no hard PyPlot/matplotlib dependency at package load)
- Unit tests + smoke tests
